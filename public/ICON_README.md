# App Icon

To add a custom house icon for the BEIT Knowledge Base app:

1. Create or find a house icon image (preferably 1024x1024 or 512x512 pixels)
2. Save it as `public/icon.png`
3. Rebuild the app

## Recommended Icon Specifications:

**For macOS:**
- Format: PNG
- Size: 1024x1024 pixels
- Background: Transparent or solid color

**For Windows:**
- Format: PNG (electron-builder will convert to .ico)
- Size: 512x512 pixels or larger
- Background: Transparent recommended

## Free Icon Resources:

- Flaticon: https://www.flaticon.com/search?word=house
- Icons8: https://icons8.com/icons/set/house
- Material Icons: https://fonts.google.com/icons?icon.query=home
